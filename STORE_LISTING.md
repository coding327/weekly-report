# Chrome 应用商店 - 上架素材

## 📦 打包文件

`chrome-extension.zip` — 直接上传到 Chrome 开发者后台即可。

---

## 🏪 商店信息

### 标题
```
AI 周报生成器 - 摸鱼神器
```

### 简短说明（132 字符以内）
```
输入这周干了什么，AI 帮你扩写成漂漂亮亮的周报。4 种风格，一键复制。不收集数据，Key 存本地。
```

### 详细说明
```
📝 还在为每周五下午憋周报发愁？

AI 周报生成器帮你搞定——输入关键词，3 秒变完整周报。

✨ 功能特点：
• 4 种周报风格：专业正式 / 简洁干练 / 详细汇报 / 互联网风
• 可选填岗位，AI 自动调整语气
• 一键复制到剪贴板，直接粘贴到飞书/钉钉/邮件
• 不满意随时换一版

🔒 隐私安全：
• 不需要注册登录
• API Key 仅保存在你本地 Chrome 浏览器
• 不会上传任何数据到第三方服务器
• 代码完全开源：github.com/coding327/weekly-report

💰 成本：
• 使用 DeepSeek API，新用户免费送 500 万 token
• 生成一篇周报约消耗 500-1000 token
• 充值 1 元能用几十次

💡 使用方式：
1. 点击浏览器右上角插件图标
2. 填入 DeepSeek API Key（获取地址：platform.deepseek.com/api_keys）
3. 输入本周工作关键词（想到什么写什么）
4. 点「生成周报」→ 完成！

🏷️ 分类：生产力工具
🏷️ 语言：中文
```

---

## 🖼️ 截图准备

需要至少 1 张 1280x800 或 640x400 的截图。建议准备 3-5 张：

1. 打开插件弹窗，展示输入界面
2. 输入示例关键词后生成结果，展示输出界面
3. 展示不同风格的周报效果

> 提示：用 Chrome 的「开发者工具 → 设备工具栏」调整到 640x400 截图效果最佳。

## 📋 上架步骤

### 第 1 步：注册开发者账号
1. 打开 https://chrome.google.com/webstore/devconsole
2. 支付一次性注册费 **$5**（约 35 元）
3. 填写开发者信息

### 第 2 步：提交扩展
1. 点击「新增项」
2. 上传 `chrome-extension.zip`
3. 填写上面准备的标题和说明
4. 上传截图（至少 1 张）
5. 选择分类：「生产力工具」
6. 语言：中文
7. 可见性：公开

### 第 3 步：等待审核
- 通常 1-3 个工作日
- 审核通过后自动上架
- 用户可以直接搜索「AI 周报生成器」找到你
